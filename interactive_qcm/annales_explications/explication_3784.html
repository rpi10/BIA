Question

Le code standard d'un transpondeur en VFR en l'absence d'instruction du contrôle est :
A- le 7000
B- le 7700
C- le 7600
D- le 7500

Explication

Les codes du transpondeur à connaître :

Code 2000 : utilisé par les aéronefs évoluant en IFR (par défaut).
Code 7000 : utilisé par les aéronefs évoluant en VFR (par défaut).
Code 7700 : utilisé par les aéronefs en situation de détresse.
Code 7600 : utilisé lorsqu'une panne des moyens de radiocommunication est détectée.
Code 7500 : utilisé pour signaler une intervention illicite (détournement).

Sur IVAO l'utilisation du code 7500 est interdite.
