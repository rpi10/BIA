Question

Quel est le type d’aile de cet avion ?
A- Delta
B- Gothique
C- Elliptique
D- En flèche

Explication

Par élimination :
- les ailes de type Delta forment un delta majuscule (Δ) vue du dessus, ce n'est pas le cas ici ;
- les ailes de type Gothique forment des arrondis, ce n'est pas le cas ici ;
- les ailes de type Elliptique forment une ellipse vue du dessus, ce n'est pas le cas ici ;
- les ailes de type en flèche forment un V vue du dessus, ce qui est un peu le cas.

Le bord d'attaque (A) n'est pas parallèle au bord de fuite (B). La réponse la plus cohérente est donc la réponse D.


Note : la description des ailes Delta se vérifie avec le nez de l'avion vers le haut, ce qui n'est pas le cas sur le document.
